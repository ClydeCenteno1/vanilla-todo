# vanilla-todo

A lightweight, persistent to-do list app built with vanilla JavaScript and Tailwind CSS — no frameworks, no dependencies.


## Features

- Add tasks via form submission
- Mark tasks as complete with strikethrough toggle
- Delete individual tasks
- Responsive dark-themed UI

## Getting Started

No build step required. Just clone and open.

```bash
git clone https://github.com/ClydeCenteno1/vanilla-todo.git
cd vanilla-todo
```

Then open `index.html` in your browser.

## What I Learned

- DOM manipulation — creating, appending, and removing elements dynamically
- Event handling — form submission, button clicks, event delegation
- `localStorage` — persisting state across sessions using JSON serialization
- Tailwind CSS utility classes for rapid UI styling
